Description

PuzzleScript is a special language created by Steven Lavelle with the intention of making game programming accessible to those without programming experience. Specifically, it uses intuitive syntax to make creating puzzle games simple.

The PuzzleScript Solver is a program that interacts with these games in order to find a solution. The focus was on the size of the state spaces considered in order to properly weigh the benefits of more optimal solutions against the use of resources.

This was made in cooperation with Chong-U Lim at MIT, who provided the framework for interacting with the PuzzleScript system.
