Description

The Glass Solid Viewer is a web application that allows the user to view models with a glass texture.

This is done by calculating and applying transparency, reflection, and refraction. In addition, the application allows the user to switch between a crystalline or non-crystalline model (smooth or flat shading), move/scale the models, change their color, and zoom in and out.

This was made using the three.js library.
